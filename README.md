# EG Personalizados — Dashboard de Gestão

Painel administrativo para gestão de pedidos, clientes e produtos da EG Personalizados.

## Tecnologias

- TypeScript · React (Next.js / Vite)
- CSS

## Como Rodar

```bash
npm install
cp .env.example .env
npm run dev
```
