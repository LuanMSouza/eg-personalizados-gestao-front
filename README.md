## 📋 Sobre

Projeto freelance desenvolvido para a **EG Personalizados**, empresa de produtos personalizados da região de Santos, SP. Dashboard para gestão interna de pedidos, clientes e produtos.

## 🚀 Como Rodar

**Pré-requisitos:** Node.js 18+ e [API de Gestão](https://github.com/LuanMSouza/eg-personalizados-gestao-back) rodando.

```bash
git clone https://github.com/LuanMSouza/eg-personalizados-gestao-front
cd eg-personalizados-gestao-front
npm install
cp .env.example .env
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
